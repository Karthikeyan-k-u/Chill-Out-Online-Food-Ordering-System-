# Chill & Out - Food Ordering System

## Short Description

A responsive, user-friendly frontend for **Chill & Out**, an online food ordering system. It features a modern menu grid with item selection and quantity adjustment, a customer detail form, and JavaScript logic to calculate and display a live order summary.

## Project Description

**Chill & Out** is a responsive online food ordering frontend website created using **HTML, CSS, and JavaScript**. The main purpose of this project is to provide a simple and user-friendly interface where customers can select food items, adjust item quantities, enter their personal details, and view the final order summary.

The website contains a modern food menu layout with items such as **Burger, Pizza, Sandwich, Fries, and Juice**. Each food item has a checkbox for selection and quantity buttons to increase or decrease the order count.

The customer details section collects the user's name, mobile number, and address. After filling in the details and selecting food items, the user can click the **Place Order** button to generate the order summary.

JavaScript is used to handle the main functionality of the project. It automatically updates the selected food quantity, validates customer details, calculates the total price, and displays a live order summary after placing the order.

This project is useful for understanding basic frontend development concepts such as responsive design, form handling, DOM manipulation, event handling, and dynamic price calculation.

## Main Functionalities

### 1. Food Item Selection

Users can select food items from the menu using checkboxes. When an item is selected, its quantity is automatically set to 1.

### 2. Quantity Adjustment

Users can increase or decrease the quantity of each food item using the plus and minus buttons.

### 3. Customer Details Form

The form collects basic customer information such as:

- Name
- Mobile number
- Address

### 4. Order Calculation

The JavaScript logic calculates the subtotal for each selected item and displays the final grand total.

### 5. Order Summary

After clicking the **Place Order** button, the website displays the complete order summary with customer details, selected food items, quantities, subtotal, and grand total.

## How to Run the Project

1. Download or clone this project.
2. Open the project folder.
3. Make sure the main file is named `index.html`.
4. Open `index.html` in any web browser.
5. Select food items, enter customer details, and place the order.
